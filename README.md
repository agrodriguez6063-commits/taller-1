# Taller 1 - Solución base de datos tienda
## Objetivo
El objetivo al solucionar este taller es extraer y analizar la información que podemos obtener por medio de esta base de datos de la tienda, como estudiante de la especialización en big data y administradora en proceso también desarrollo el objetivo de aplicar consultar que me permiten dar respuesta a preguntas reales de este y cualquier otro negocio al que se quiera aplicar.
## Desarrollo
Al momento de realizar este taller pude comprender las diferentes estructuras de la base de datos, logré interpretar correctamente las preguntas y transformar los datos en información útil que van a servir en la toma de decisiones.
Logré analizae ventas, comportamientos de clientes, uso de cupones y rendimiento de productos. 
## Conclusiones
1. El SQL es una herramienta muy útil para hacer analisis al un negocio.
2. El análisis va más alla de lo que es técnico, ya que al hacer ejercicios como clientes de riesgo o clasificaión por nivel de gasto demuestran cómo SQL puede ser un apoyo al momento de tomar decisiones estratégicas.
3. La orgnización y los documentos son tan importantes como los códigos que se desarrollaron, ya que si se cuenta con un repositorio estructurado y claro, no solo nos ayuda a ser agiles con el manejo sino que demuestra buenas prácticas en desarrollo de bases de datos.
